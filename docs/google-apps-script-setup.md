# Google Apps Script Setup

1. Open your Google Sheet.
2. Select `Extensions > Apps Script`.
3. Replace the default editor contents with `apps-script/Code.js`.
4. Save.
5. Reload the spreadsheet.
6. Use `HTB Bot > Setup sheets`.
7. Use `HTB Bot > Set HTB token`.
8. Use `HTB Bot > Test HTB token`.
9. Use `HTB Bot > Baseline current solves`.
10. Use `HTB Bot > Install 10-min detector`.

Google will ask for authorization the first time the script uses spreadsheet, URL fetch, or trigger permissions.
